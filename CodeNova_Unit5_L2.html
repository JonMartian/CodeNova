Ano ang Set?

Ang set ay parang bag ng mga bagay kung saan hindi pinapayagan ang duplikado at hindi mahalaga ang pagkakasunod-sunod. Kapaki-pakinabang ang sets kapag ang mahalaga lang ay ang mga natatanging items o kung gusto mong mabilis na i-check kung nasa loob ang isang item.
